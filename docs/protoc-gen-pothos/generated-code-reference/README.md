# Generated Code Reference

Reference documentation for the TypeScript code generated by protoc-gen-pothos.

## Choose Your Runtime

Select your protobuf runtime for the complete generated code reference:

| Runtime | Description |
|---------|-------------|
| **[ts-proto](./ts-proto.md)** | Uses `$type` property for type discrimination, oneof fields accessed directly |
| **[protobuf-es-v1](./protobuf-es-v1.md)** | Uses `instanceof` for type discrimination, includes `$toProto` helper functions |
| **[protobuf-es](./protobuf-es.md)** | Uses `isMessage()` and `create()` functions, Schema-based API |
